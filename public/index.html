Timestamp Microservice

The Timestamp Microservice helps you to get a JSON object with a Unix Timestamp code.
You can provide a custom date or access /api to get the current date value.

Input Example:

/api/2023-12-31

or

/api/1703980800000

Output:

{"unix":1703980800000,"utc":"Sun, 31 Dec 2023 00:00:00 GMT"}

Try it out: /api/2023-12-31
